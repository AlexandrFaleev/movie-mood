# Учебный проект "Mood movies" на React JS

## Основная идея

Mood Movies - приложение, позволяющее пользователю составлять список фильмов, которые он хотел бы посмотреть или уже посмотрел, с привзякой к эмоциям, которые зритель испытывает при просмотре.

## Проектное задание №2

### Что сделано?

- Добавлена работа с API (на базе json-server);
- Ранее useEffect использовался для работы с LocalStorage, теперь же он используется для API;
- Добавлена маршрутизация (для каждого фильма есть возможность перейти на страницу с его детализацией;
- Общее состояние - создан контекст, минимизирован prop-drilling
